Portfolio

Below is a list of projects I have worked on:

  1. Fighting Game (Solo Project): A Java Command Line rock, paper, scissors clone.

    • Features a simple text-based interface.

    • Includes basic error handling for user input.

    • Utilizes Java's built-in graphics library for rendering.

    • Features realistic physics for ball movement and collision detection.

    • Built in object removal tool for when the balls fall into the pockets.
